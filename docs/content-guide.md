# Інструкція редактора

Увійдіть через Cloudflare Access. Створіть категорію, потім draft-публікацію: український title/body обов'язкові. Перевірте Markdown preview, alt/credit/license для фотографій, slug та SEO. Publish робить матеріал публічним; archive прибирає його з публічних списків. Для English заповніть окремі поля: не позначайте англійську версію published без title і body. Зміна slug повинна створити перевірений redirect.
